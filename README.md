
# Phigros Recorder 介绍

Better rendering of Phigros chart.
更好的渲染 Phigros 谱面。

## Features 功能

- Chart Revealed 谱面揭秘
- CRF bitrate control CRF码率控制
- Support HEVC 支持 HEVC 编码
- Lossless Audio 无损音频
- Disable Loading Scene 禁用加载场景
- ...

## Installation 安装

Check [Releases](https://github.com/2278535805/phigros-recorder/releases)

## Manual build 手动构建

Build

```bash
cargo tauri build
```
